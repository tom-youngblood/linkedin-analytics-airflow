Initial steps: Copy the files in this repo into new folder. 

1.  Astro CLI (astronomer.io)
- install Astro CLI
- Go to terminal, command: astro dev init
- Gives us the entire best practices airflow CLI
- Execute astro dev start --> Gets airflow up and running on my machine
